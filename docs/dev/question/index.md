## 解决 img 资源使用 v-bind 动态引入的问题

详细问题：

原理：使用`import`将相对路径资源导入，`import`过程中会自动解析相对路径为目标资源的绝对路径

```vue
<script setup>
import svg from "@/assets/icons/s.svg";
</script>

<template>
  <img :src="svg" />
</template>
```
