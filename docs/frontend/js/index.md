# 简介

## 为什么说js是单线程的语言

- 设计初衷：JavaScript最早是作为浏览器的脚本语言而设计的，用于处理网页交互和动态内容。为了简化语言和避免复杂性，JavaScript引擎被设计为单线程的
- UI渲染和响应：JavaScript在浏览器中主要负责处理UI渲染和响应用户交互，包括DOM操作、事件处理、样式计算等。如果JavaScript是多线程的，多个线程同时对DOM进行操作可能会导致不一致和冲突。
- 避免竞态条件：多线程并发执行时，可能会出现竞态条件（Race Condition）和数据不一致的问题。为了避免这些问题，JavaScript选择了单线程的执行模型，确保任意时刻只有一个线程在执行JavaScript代码。
